# 🚀 AI-Based Monitoring Dashboard with Prometheus

![Dashboard Preview](https://via.placeholder.com/1000x400?text=AI+Monitoring+Dashboard)

## 📌 Project Overview

This project is an **AI-powered system monitoring dashboard** that tracks real-time system metrics and detects anomalies using Machine Learning.

It combines **DevOps + AI + Cloud Deployment** into a single intelligent monitoring solution.

---

## 🔥 Features

- 📊 Real-time CPU & Memory Monitoring  
- 🤖 AI-based Anomaly Detection (Isolation Forest)  
- 📡 Prometheus Metrics Integration  
- 🎨 Premium Dashboard UI (Modern SaaS Design)  
- ☁️ Cloud Deployment using Render  

---

## 🧠 Tech Stack

- **Backend:** Python, Flask  
- **Machine Learning:** Scikit-learn (Isolation Forest)  
- **Monitoring:** Prometheus  
- **Frontend:** HTML, CSS, Chart.js  
- **Deployment:** Render  
- **Tools:** Git, GitHub
